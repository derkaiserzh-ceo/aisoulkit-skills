# AI SOUL KIT — Claude Code Plugin

**Spezialisierte Agenten direkt in Claude Code.**  
Kein Copy-Paste. Einfach `/axiom` tippen — fertig.

---

## Installation

```bash
claude plugin marketplace add derkaiserzh-ceo/aisoulkit-skills
claude plugin install aisoulkit
```

Danach verfügbar:

| Befehl | Agent | Was er tut |
|--------|-------|-----------|
| `/axiom` | AXIOM | Strategische Entscheidungen & Projekt-Execution |
| `/vance` | VANCE | Content CEO — Posts, Artikel, Newsletter |
| `/closer` | CLOSER | Sales Agent — Outreach, Follow-up, Angebote |
| `/builder-kit` | SOUL Builder | Eigenen Agenten in 10 Min bauen |

Skills aktivieren sich auch **automatisch** — Claude erkennt aus dem Kontext wann AXIOM oder VANCE passt.

## Was macht ein SOUL anders?

Ohne SOUL antwortet Claude sofort — egal wie komplex die Frage.  
Mit SOUL klassifiziert er zuerst (E1 Quick → E4 Kritisch), aktiviert passende Denk-Modi und prüft seinen Output gegen Quality-Gates bevor er liefert.

```
Ohne SOUL:  "Hier sind 5 Optionen für dich..."
Mit AXIOM:  EFFORT-LEVEL: E3
            EMPFEHLUNG: Option B
            RISIKO: ...
            NAECHSTER SCHRITT: Wer + Was + Bis wann
```
